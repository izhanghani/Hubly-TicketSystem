Everything you need to
manage IT support

From ticket creation to SLA monitoring, Hubly provides a complete toolkit for your help desk.

01

Ticket Management

Create, track, and manage tickets with full status workflow: open, in progress, resolved, and closed.

02

SLA Tracking

Automated SLA monitoring with business hours, escalation rules, and real-time breach detection.

03

Role-Based Access

Admin, Agent, Supervisor, and User roles with granular permissions and department-based visibility.

04

Workflow Automation

Auto-route tickets to departments or specific agents based on customizable matching rules.

05

Real-Time Notifications

Instant alerts via Socket.IO and optional email notifications through SMTP configuration.

06

Dashboard & Reports

Real-time statistics, charts, agent performance metrics, and CSV data export functionality.

07

File Attachments

Drag-and-drop file uploads with validation and organized attachment management per ticket.

08

Dark & Light Mode

Built-in theme toggle with smooth transitions, system preference detection, and persistent choice.

09

Audit Logs

Full audit trail for every action with timestamps, user attribution, and exportable log data.

Quick Start

Get Hubly running on your machine in under a minute. v2.0 now supports Docker and cross-platform start scripts.

# Clone & install
git clone https://github.com/izhanghani/Hubly-TicketSystem.git
cd Hubly-TicketSystem
npm install

# Start (interactive menu)
node start.js

# Or development mode directly
npm run dev
# Option 1: Double-click any of these:
start.bat
run.ps1
Start App.vbs

# Option 2: From terminal
node start.js

# Or directly:
npm install
npm run dev
# Make executable & run:
chmod +x start.sh
./start.sh

# Or use the cross-platform Node.js script:
node start.js

# Or directly:
npm install
npm run dev
# Using Docker Compose (recommended):
docker compose up

# Or build & run manually:
docker build -t hubly .
docker run -p 3000:3000 hubly
