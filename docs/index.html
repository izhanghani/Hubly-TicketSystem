Real-time statistics, charts, agent performance metrics, and CSV data export.

Quick Start

Get Hubly running in under a minute:

# Clone & install
git clone https://github.com/izhanghani/Hubly-TicketSystem.git
cd Hubly-TicketSystem
npm install

# Start development server
npm run dev

Then open http://localhost:5173 — Login: admin / admin123

Built with Node.js, Express, SQLite, Socket.IO, and Vite
