Everything you need to
manage IT support

From ticket creation to SLA monitoring, Hubly provides a complete toolkit for your help desk.

01

Ticket Management

Create, track, and manage tickets with full status workflow: open, in progress, resolved, and closed.

02

SLA Tracking

Automated SLA monitoring with business hours, escalation rules, and real-time breach detection.

03

Role-Based Access

Admin, Agent, Supervisor, and User roles with granular permissions and department-based visibility.

04

Workflow Automation

Auto-route tickets to departments or specific agents based on customizable matching rules.

05

Real-Time Notifications

Instant alerts via Socket.IO and optional email notifications through SMTP.

06

Dashboard & Reports

Real-time statistics, charts, agent performance metrics, and CSV data export.

07

File Attachments

Drag-and-drop file uploads with validation and organized attachment management per ticket.

08

Dark Mode

Built-in dark and light theme toggle with smooth transitions and system preference detection.

09

Audit Logs

Full audit trail for every action with detailed timestamps, user attribution, and exportable logs.

Quick Start

Get Hubly running on your machine in under a minute.

# Clone & install
git clone https://github.com/izhanghani/Hubly-TicketSystem.git
cd Hubly-TicketSystem
npm install

# Start development
npm run dev
# Just double-click:
Start App.vbs

# Or from terminal:
npm install
npm run dev
# Coming soon — Docker support in progress
docker pull hubly/hubly
docker run -p 3000:3000 hubly/hubly
